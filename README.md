# Project Management - 21/22 (FCTUC)

O objetivo deste trabalho é aprimorar capacidades em atividades de gestão e desenvolvimento de um projeto real, a pedido da organização Instituto Pedro Nunes (IPN), representada pelo consultor e cliente Alcides Manuel de Almeida Marques.

O Projeto destina-se à criação e desenvolvimento de uma solução Web, que serve para a gestão de ativos do IPN.
